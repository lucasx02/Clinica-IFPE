# Sistema de Gestão – Clínica para Surdos

## Descrição
Sistema web para gerenciamento de uma clínica especializada em atendimento
a pacientes surdos, contemplando cadastro de pacientes, profissionais,
agendamento de consultas e controle de intérpretes de Libras.
